Welcome to the Hawaii Weather Data API

Available API Routes

  1. /api/v1.0/precipitation - Returns precipitation data for the last year.
  2. /api/v1.0/stations - Lists all weather stations.
  3. /api/v1.0/tobs - Returns temperature observations of the most active station for the last year.
  4. /api/v1.0/start_date - Returns temperature statistics (temp max, min, avg) from a given start date.
  5. /api/v1.0/start_date/end_date - Returns temperature statistics (temp max, min, avg) between a given start date and end date.
